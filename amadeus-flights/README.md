# Amadeus Flights Skill ✈️

An OpenClaw skill for searching flights via the Amadeus Self-Service API.

## Features

- **Flight Search** — One-way and round-trip searches
- **Flexible Filters** — Class, nonstop, max price, multiple passengers
- **Airport Lookup** — Find IATA codes by city/airport name
- **Price Confirmation** — Verify offer pricing before booking

## Part of TravelClaw

This skill was built as part of the [TravelClaw](https://github.com/lyra-claw/travelclaw) project — exploring AI-assisted travel planning with OpenClaw.

## Setup

1. Get free API credentials at https://developers.amadeus.com/self-service
2. Set environment variables:
   ```bash
   export AMADEUS_API_KEY="your-key"
   export AMADEUS_API_SECRET="your-secret"
   ```
